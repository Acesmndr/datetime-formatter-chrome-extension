DateTime Formatter (Settings)
Convert all the date and time in current page to specified datetime format
The settings can be updated again by clicking on the extension icon and pressing the options menu item.
What's new in v1.2.0
* Add support for a new date type.
* Add tooltip for highlighted converted date to show original date that got converted.
* Add valid date range support so that only a range of +/-100 years is converted when enabled.
* Allocate memory only while conversion is done
